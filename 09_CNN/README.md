**CNN Image Classification on CIFAR Dataset**
This project implements a Convolutional Neural Network (CNN) to classify images from the CIFAR dataset using Keras/TensorFlow. CIFAR is a standard benchmark dataset containing 60,000 32x32 color images in 10 classes (CIFAR-10)
Dataset

CIFAR-10: 60,000 32x32 color images, 10 classes:

50,000 training images

10,000 test images

Classes: airplane, automobile, bird, cat, deer, dog, frog, horse, ship, truck
